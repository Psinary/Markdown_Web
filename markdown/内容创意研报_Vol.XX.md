# 关于研报：

### 研报是什么？

它是我个人学习的一种方法，属于信息清洗的中间产物，最终目的是升级认知，系统化沉淀知识。
研报主要涉及互联网和新旧媒体等方面，例如，内容，设计，营销，品牌，产品等领域。
